%---------- Abstract ----------------------------------------------------------

\Abstract{Microservice is een nieuwe manier om services te maken. Deze manier kan de huidige integration patterns vervangen bij het maken van een stabielere applicatie. Uit onderzoek moet blijken hoe deze technologie kan toegepast worden op het order-to-cash proces in SAP. SAP is een veel gebruikte software. We gaan kijken hoe microservices een order-to-cash proces beïnvloedt. Zodat de services geen hinder ondervinden bij het falen of vertragen van één of meerdere services. Dit zal een vergelijkende studie zijn. 
}
